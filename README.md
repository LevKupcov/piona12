# piona12

Repository with two separated projects:

- `old-company-enricher/` - the original Company Enricher project from the previous repository state.
- `crm-bp-search/` - Bitrix24 CRM business-process activity for searching CRM items by optional criteria.

In `crm-bp-search`, search criteria can be filled independently. Empty criteria are ignored, and at least one filled criterion is required before a search can run.
